Qui默认设置的圆角有：

	.bdr6 {border-radius: 6px;}

	</div>

object-fit:控制图片

img-fill:object-fit: fill;
img-contain:object-fit: contain;
img-cover:object-fit: cover;
img-none:object-fit: none;
img-scale:object-fit: scale-down; 
	.img-fill { object-fit: fill; }
	.img-contain { object-fit: contain; }
	.img-cover { object-fit: cover; }
	.img-none { object-fit: none; }
	.img-scale { object-fit: scale-down; }

创意图片排列

注意：创意图片只能以背景图片的形式插入，不可用img标签

	单个图片：ui-img-1 ，两个图片：ui-img-2 ，三个图片：ui-img-3 ，四张以上图片都用三列排法，可以留空跳格实现四格
	<div class="ui-img-wrap">
	<div class="ui-img-1" style="background-image:url(../img/01.jpg)"></div>
	</div>
	
	不同数量排列方式不一样，展示效果也就不一样 ，例如：3张排列就有3中排列
	3= 2+1 = 1+2 = 1+1+1
	
	特列就是这个3=2+1的排列，下面是复用代码
	<div class="ui-img-wrap mtb20">
	<div class="ui-img-31" style="background-image:url(../img/01.jpg)"></div>
	<div class="ui-img-32" style="background-image:url(../img/01.jpg)"></div>
	<div class="ui-img-32" style="background-image:url(../img/01.jpg)"></div> 
	</div>
